Chapter 2. Installation

2.1. Common configuration for all platforms

Note that KTiny is intended to work with TinyERP 4.2 server or newer.

Although KTiny tries to work without any server modification, the ktiny server module is necessary for proper use of attachments. Please, simply install this module that can ve found in server-modules directory, as you would with any other module, and attachments should work smoothly.

This module also provides client side configuration options that can be found at Administration -> Configuration -> KTiny -> Settings.
