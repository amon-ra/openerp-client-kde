4.4. Views

A view in TinyERP is a graphical representation of a data set. Obvious views that come to mind are form and list plus currently KTiny also offers a chart view. All of them are available in the GTK and Web user interfaces. A calendar view is being worked on these two clients too, and KTiny we'll add it in the future. Here we want to explain how a developer can add another view to fit his/her needs.

The three views can be found in different subdirectories inside bin/widget/view. To summarize we'll say that in order to create a new view you need to create a new directory with a __terp__.py file and a couple of classes. One inheriting from AbstractView and another one inheriting from AbstractParser (whose interfaces can be found at bin/widget/view).
