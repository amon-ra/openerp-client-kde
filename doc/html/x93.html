2.4. Mac OS

In this section we're aggregating information on how to make Koo work on the Mac platform.

Needed software:

  • Qt4 for Mac (http://trolltech.com/developer/downloads/qt/mac)

  • From pythonmac.org: python 2.5 dmg

    • python 2.5 dmg

    • matplotlib dmg

    • numpy dmg

    • pytz dmg

  • From http://www.riverbankcomputing.com (download + compile): sip and pyqt4

  • From http://www.koders.com copy dextras.py into koo/

  • From http://www.freedesktop.org:

    • pkg-config compile

    • export PATH=/usr/local/bin:$PATH

    • libxml (does not find Python.h)

    • dbus

  • agg2.5 - configure errors

    • sdl: configure --prefix=/usr

    • libtoolize

  • port (problems building)

    • sudo /opt/local/bin/port upgrade all

    • sudo /opt/local/bin/port install

      • autoconf

      • automake

      • libtool

      • autoconf
